# ADVACC — Portfolio Website

## Project Overview
- **Name**: ADVACC Portfolio
- **Goal**: Professional portfolio website for ADVACC — an accounting & HR management & database systems development professional
- **Tech Stack**: Next.js 16 + React 19 + TypeScript + Tailwind CSS 4 + Cairo/Inter/Tajawal fonts

## URLs
- **Live Sandbox**: https://3000-ipve06bnimdtqw28ynb24-2e77fc33.sandbox.novita.ai

## Features
- Dark theme by default with light mode toggle
- "بسم الله الرحمن الرحيم" banner bar above header
- Responsive mobile dropdown menu (hamburger)
- Hero section with animated profile circle and counter stats
- 5 work experience entries in timeline format
- 6 skill categories with detailed items
- 26+ projects filterable by ministry (تربية, أوقاف, ثقافة, صحة)
- 3 goal periods (short, medium, long term)
- Contact form via FormSubmit.co to advacc2000@gmail.com
- Scroll progress bar, back-to-top button, scroll reveal animations
- Fully responsive on all screen sizes (1400px to 360px)
- RTL Arabic layout with Cairo font
- Zero console errors

## Pages
| Route | Description |
|-------|-------------|
| `/` | Hero section with profile, stats, CTA |
| `/experience` | Work experience timeline (5 jobs) |
| `/skills` | Professional skills (6 categories) |
| `/projects` | 26+ protected projects with filters |
| `/goals` | Career goals (short/medium/long term) |
| `/contact` | Contact form, phone, email, WhatsApp |
